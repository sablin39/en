让直方图“连续”起来：直方图与核密度估计

直方图是密度的一个相当粗糙的估计量，它的外观对单元格的数目和位置都是敏感的，x,y轴的变动会显著影响其观感（像个渣男, 做出点小小的改变就欺骗了你的感性）。因此引入核密度估计量让它更像一个概率密度函数，本质上是在一个数据点上给前后的数据以不同权重，形成相对平滑的曲线（感性上可认为与卷积略有相似）。基于 dataset 的核密度估计量可以写作：

其中K一般是标准正态密度函数即，b被称为带宽，决定了估计量的分辨率。

fit(X[, y, sample_weight]) Fit the Kernel Density model on the data.
get_params([deep]) Get parameters for this estimator.
sample([n_samples, random_state]) Generate random samples from the model.
score(X[, y]) Compute the total log-likelihood under the model.
score_samples(X) Compute the log-likelihood of each sample under the model.
set_params(**params) Set the parameters of this estimator.
